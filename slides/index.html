Riešenie

Jedna verzia. Celá história.

Git = časová os projektu. Každý bod je uložený stav, ku ktorému sa vieš vrátiť.
Tri slová, ktoré sa pletú
Git
nástroj/systém
na tvojom počítači
GitHub
miesto v cloude,
kde repo zdieľame
(ako Drive pre projekt)
GitKraken
appka — pohodlné
okno do gitu
Nie sú to to isté — sú to tri rôzne veci, ktoré spolupracujú.
Repozitár

Projekt + jeho celá história, na jednom mieste.

Priečinok s tvojimi súbormi a zároveň pamäť všetkých zmien. Skrátene „repo“.
Commit

Bod na osi — nie obyčajný „save“.

✓ Hovorí čo sa stalo

Add my introduction
Fix typo in README
Update workshop notes

✗ Nepovie nič

changes
update
stuff

Commit = uložený balíček zmien + krátky popis čo a prečo.
