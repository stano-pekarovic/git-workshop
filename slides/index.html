Riešenie

Jeden zdroj pravdy — s celou históriou.

prvá verzia
pridaná stránka
oprava textu
dnešný stav
Každý bod je uložený stav, ku ktorému sa vieš kedykoľvek vrátiť.
Tri pojmy, ktoré sa pletú
Git
systém na kontrolu verzií
na tvojom počítači
GitHub
cloudová služba na ukladanie repozitárov
a zdieľanie s tímom
GitKraken
grafické rozhranie nad gitom
appka na počítači
Nie sú to to isté — sú to tri rôzne veci, ktoré spolupracujú.
Repozitár

Projekt + jeho celá história, na jednom mieste.

Priečinok s tvojimi súbormi a zároveň pamäť všetkých zmien. Skrátene „repo“.
Commit

Bod na osi — nie obyčajný „save“.

✓ Hovorí čo sa stalo

Add my introduction
Fix typo in README
Update workshop notes

✗ Nepovie nič

changes
update
stuff

Commit = uložený balíček zmien + krátky popis čo a prečo.
